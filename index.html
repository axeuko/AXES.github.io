Autonomous Land Exloration
(ALEX)

In this project i designed and built an AGV for the purpose of autonomously exploring previously unmapped/undiscovered areas. To achieve this, after the construction of the vehicle's chasis, i built an on board server that records and transmits live video from the vehicle's on-board camera and also transmits data from other on-board sensors (that could later be added or removed). In addition to this i built a webaapp where the vehicle's motion can be controlled remotely to serve as the 'Ground Control' of the vehicle. This project was achieved using Python, Flask, C++, HTML, CSS and Javascript.
